Particle Swarm Optimization

for hyper-parameter tuning
0 1

This visualization showcases the process by which an evolutionary algorithm searches for the global optima of a multi-parameter problem.

A common use for this algorithm is the optimization of Machine Learning (ML) model hyper-parameters. These are the settings that the drive the learning rate and accuracy of the model.

For example, a Deep Neural network has thousands of possible parameter combinations. The particle swarm is not guaranteed to find the absolute best one, but it will converge to one of the best combinations in a relatively short amount of time.
